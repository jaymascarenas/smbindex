# SMB Index

####Data visualization application wth a JSON driven data set by month.


##### Libraries used

- jQuery
- Bootstrap
- C3
- datamaps.js
- guage.js
- MonthPicker.js
- clipboard.js


#####Gulp task and build package

- Install npm packages 

```npm install```

- Install bower packages

```bower install```

- Start Gulp server

```gulp serve```

- Build Gulp dist package

```gulp build```
